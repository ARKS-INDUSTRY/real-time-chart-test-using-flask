Command lines:
<br>
>
 $   mkdir flask 
<br>
> $  cd flask
<br>
> $  python3 -m venv venv
<br>
<br>
to activate venv:
<br>
<br>
>$  . venv/bin/activate
<br>
>$  pip install Flask
<br>
>$  export FLASK_APP=app.py
<br>
>$  flask run
<br><br>

to run on the server publiclly available:

<br>
<br>
>$  flask run --host 0.0.0.0
